# AVR_Washing_Machine


모터 : 2초 정회전 2초 역회전 반복 (단, 탈수 시에는 정회전)
LED  : 각 상태에 맞게 LED 점등
FND  : 남은 시간 출력
LCD  : 현재 동작 상태 표시

동작 영상 1 ( 표준 세탁 모드 )
